Talk Title

Machine Learning for Scientific Discovery, with Examples in Fluid Mechanics

Abstract

This talk discusses key challenges and opportunities in using machine learning for nonlinear system identification. Prof. Brunton will show how machine learning can develop accurate dynamical models for complex natural and engineered systems.

He will emphasize interpretable and generalizable approaches such as the sparse identification of nonlinear dynamics (SINDy) framework — which balances model simplicity and accuracy while avoiding overfitting.

Several benchmark problems in dynamical systems and fluid mechanics illustrate how system identification techniques can be evaluated in diverse settings.

Because fluid dynamics underpins transportation, health, and defense systems, the talk will highlight the importance of data-driven models that remain interpretable, explainable, generalizable, and consistent with physics.
